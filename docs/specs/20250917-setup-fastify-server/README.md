# Fastify Server 設置專案

## 專案概述

本專案旨在建立一個基於 TypeScript 和 Fastify 的現代化 Web 伺服器應用程式。專案採用最佳實踐的架構設計，提供高性能、可擴展且易於維護的伺服器解決方案。

## 專案目標

- 建立一個完整的 Fastify 伺服器架構
- 實現模組化和可重用的程式碼結構
- 提供完整的 API 文檔和規範
- 確保程式碼品質和測試覆蓋率
- 建立持續整合和部署流程

## 技術棧

- **運行時環境**: Node.js 18+
- **程式語言**: TypeScript 5+
- **Web 框架**: Fastify 4+
- **測試框架**: Vitest, Jest
- **建置工具**: esbuild, Rollup
- **程式碼品質**: ESLint, Prettier
- **容器化**: Docker
