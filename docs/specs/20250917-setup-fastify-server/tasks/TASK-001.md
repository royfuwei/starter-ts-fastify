# TASK-001: 環境準備

## 任務概述
建立完整的開發環境，確保所有必要的工具和軟體都已正確安裝和配置。

## 任務模式
**Code 模式** - 用於具體的環境配置和工具安裝

## 任務目標
- 安裝並配置 Node.js 18+
- 安裝並配置 TypeScript 5+
- 配置 VS Code 開發環境
- 安裝 Git 版本控制
- 設置完整的開發工具鏈

## 具體實作步驟

### 1. Node.js 18+ 安裝
- [ ] 檢查當前 Node.js 版本
- [ ] 下載並安裝 Node.js 18+ LTS 版本
- [ ] 驗證 Node.js 和 npm 安裝
- [ ] 配置 npm 全局包路徑

### 2. TypeScript 5+ 安裝
- [ ] 全局安裝 TypeScript 5+
- [ ] 驗證 TypeScript 版本
- [ ] 配置 TypeScript 全局選項
- [ ] 安裝 TypeScript 相關工具

### 3. VS Code 開發環境配置
- [ ] 安裝 VS Code 編輯器
- [ ] 安裝必要的擴展插件：
  - ESLint
  - Prettier
  - TypeScript Language Support
  - GitLens
  - Path Intellisense
- [ ] 配置 VS Code 工作區設置
- [ ] 建立代碼片段模板

### 4. Git 版本控制設置
- [ ] 安裝 Git
- [ ] 配置 Git 用戶信息
- [ ] 設置 SSH 金鑰
- [ ] 配置 Git 别名和工具
- [ ] 建立全局 .gitignore 文件

### 5. 開發工具鏈設置
- [ ] 安裝開發工具：
  - nodemon（自動重啟）
  - concurrently（並行運行）
  - rimraf（文件清理）
- [ ] 配置開發腳本
- [ ] 建立開發環境驗證腳本

## 預期交付物
- 完整的開發環境配置
- 工具安裝驗證腳本
- 環境配置文檔
- 開發工具使用指南

## 測試要求
- [ ] Node.js 版本 >= 18.0.0
- [ ] npm 版本 >= 8.0.0
- [ ] TypeScript 版本 >= 5.0.0
- [ ] Git 配置正確
- [ ] VS Code 擴展插件正常工作
- [ ] 開發工具腳本可執行
